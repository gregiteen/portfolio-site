# Bauhaus Minimalist Design Spec

This framework establishes an uncompromising, technical interface inspired by Bauhaus geometry and high-precision file-native systems engineering. It is designed specifically for Greg Iteen's professional portfolio archive.

## Architectural Rationale

1. **Stark Geometry**: The visual system utilizes harsh gridlines, heavy solid borders (2px and 3px), and zero border-radius to align with the core philosophy of "form follows function".
2. **Minimalist Chromatic Scheme**: Relying almost entirely on an editorial dark palette, the design introduces tension through pure Bauhaus Red (`#E22B22`) as an operational indicator rather than static decoration.
3. **Typography as Structure**: Large, bold sans-serif lettering functions as structural dividers, while raw, monospace indicators present operational details directly.

## Spacing & Mobile Architecture

The theme utilizes a strict, fluid layout utilizing `min-width` media queries to support responsive scaling. Tap targets are locked to a minimum of `44px` with deliberate padding profiles to ensure flawless mobile ergonomics without softening the geometry.
