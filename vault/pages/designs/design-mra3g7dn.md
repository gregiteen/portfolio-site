# Design Spec: Titanium Sovereign

### 1. Visual References
* **Dieter Rams' Braun T3 Pocket Radio & Physical Machinery Molds**: The chassis is not polished; it is matte-blasted, heavy, and structured. This translates to an interface built on absolute physical bounds, thick 2px titanium borders, and zero-radius corners.
* **Early Nuclear Power Plant Control Consoles**: Functional grids, distinct physical panels, high-contrast block warnings, and stark labels. It is designed to be read under high-stress scenarios where clarity overrides decorative fluff.
* **Technical Specifications Bulletins (Deutscher Normenausschuss)**: Tightly tracked, dense DIN-like typography, left-aligned layout matrices, and safety-orange classification stamps that signify high-priority metadata.

### 2. Typography Rationale
We pair two Google Fonts that generally scream with distinct architectural purposes:
* **Saira Extra Condensed** (Header/Label Typeface): Set in heavy weights with tracking as tight as `-0.03em`. It mimics compressed warning placards and die-cut metal stampings. It forces titles to act like physical, load-bearing beams across the top of sections.
* **Albert Sans** (Body/System Typeface): A modern European geometric grotesk. It is neutral, precise, and highly legible even when rendered inside miniature metadata grids. Tracking is pulled tight in subheads to reference classic Swiss typographic posters.

### 3. Color Palette
* `charcoal-base` (`#0F1011`): A cold, lightless black resembling oxidized industrial steel plates.
* `titanium-gray` (`#2B2D31`): A heavy, mid-tone structural gray used for framing, borders, and systemic divisions.
* `alloy-subtle` (`#8A8E94`): An inactive, low-contrast gray for secondary labels, grid markers, and systemic codes.
* `inert-white` (`#F3F4F6`): A crisp, stark white that cuts clean through the darkness like laser engraving.
* `safety-orange` (`#FF4500`): The only color accent. Reserved strictly for warnings, primary interactive elements, active states, and structural system stamps.

### 4. Layout Strategy & Signature Device
Our signature structural device is the **Precision Modular Matrix (PMM)**. The viewport is treated as a single machined instrument face. Every page component is locked into a rigid 2px matte-gray frame. There are no loose elements. 

Every module starts with an industrial header block containing a live system timestamp, a static file-size block size label (`SYSTEM_ADDR // 0x4096`), and a bold classification label. 

To break standard templates, the navigation is styled as an industrial control matrix mounted horizontally at the top, complete with an illuminated orange active channel. The hero section is not a typical visual splash but rather an "Operating Manual Thesis"—bold technical assertions positioned directly next to an interactive, heavy-bordered schematics viewer loaded with Greg's visual generator feed.
