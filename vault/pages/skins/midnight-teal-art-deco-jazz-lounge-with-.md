## The Concept: Midnight Brass Art-Deco Jazz Lounge

This system is built specifically for Greg Iteen's portfolio of local-first, file-native AI projects. It steps away from current generic "bento grids" and "dark-mode neon" templates to embrace a highly customized visual identity inspired by a **midnight teal art-deco jazz lounge with brass accents and technical machinery instrumentation**.

### 1. Architectural Geometry (The Chassis)
- **Ziggurat Framing**: Utilizing pure CSS clip-path and double nested borders to build stepped corners, reminiscent of both classic Art Deco buildings and industrial machinery control panels.
- **Dual-Rail Header**: Main navigation links are floating inside physical double-brass horizontal rails mimicking dial guides on vintage shortwave radios.

### 2. Typography & Hierarchy
- **Display Typography**: Large structural titles use the ultra-geometric uppercase 'Forum' font with extremely wide tracking to mimic vintage brass nameplates.
- **Readouts & Microdata**: All technical metrics, stats, and metadata are set in 'JetBrains Mono' to reflect Greg's file-native, engineering-driven workflow.

### 3. Kinetic Interactions
- **Vacuum Tube Warming**: Elements fade-in and hover with an amber glow inspired by high-end vacuum tube filaments (`var(--vacuum-glow)`).
- **Analog Dial Sound Effect**: Standard clicking on theme toggles and navigational controls generates an organic synthetic analog sound click via the Web Audio API.
- **Art-Deco Scroll Telemetry**: An animated geometric SVG radio tower serves as a visualizer indicating file system access speeds.
