Every rule, correction, and fact an agent learns becomes a validated Markdown
node in a local vault — no external database, ever. Semantic search runs on
local dense embeddings, compiled instruction surfaces inject active rules into
any IDE agent, and a background daemon researches topics while you sleep.
This site's build pipeline is registered in its project brain.
